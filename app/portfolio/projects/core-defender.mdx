A strategic tower defense game where players must defend their core from increasingly difficult waves of enemies by strategically placing and upgrading defensive structures.

## Overview

Core Defender challenges players to think strategically as they place towers, manage resources, and adapt their defense strategy to survive against waves of enemies with different abilities and weaknesses.

## Features

- Multiple tower types with unique abilities
- Wave-based enemy spawning system
- Resource management and economy
- Tower upgrade system
- Different enemy types with varying behaviors
- Progressive difficulty scaling
- Strategic gameplay mechanics

## Technologies Used

- JavaScript (ES6+)
- HTML5 Canvas API
- CSS3 for styling
- Game loop implementation
- Pathfinding algorithms

## Inspirations

- There Is No Game
- Blackbox
- Pony Island
- Her Story
