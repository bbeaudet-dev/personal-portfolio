An intelligent chatbot designed to entertain and challenge users with riddles, puzzles, and interactive conversations while providing hints and guidance.

## Overview

Riddlemaster is a conversational AI that specializes in riddles and brain teasers. It can adapt its difficulty level, provide hints when needed, and engage users in extended puzzle-solving sessions.

## Features

- Dynamic riddle generation and presentation
- Adaptive difficulty system
- Hint system for challenging puzzles
- Natural language conversation capabilities
- Score tracking and progress monitoring
- Multiple riddle categories and themes

## Technologies Used

- Python
- Natural Language Processing libraries
- API integration for enhanced responses
- JSON data structures for riddle storage
- Regular expressions for pattern matching
