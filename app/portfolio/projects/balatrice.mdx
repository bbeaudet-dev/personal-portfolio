A roguelike deck-building card game inspired by Balatro, featuring unique card mechanics, strategic decision-making, and procedurally generated challenges.

## Overview

Balatrice combines elements of traditional card games with modern roguelike progression, creating an engaging experience where players build powerful card combinations and face increasingly difficult challenges.

## Features

- Deck-building mechanics with card collection
- Procedurally generated runs and challenges
- Unique card effects and synergies
- Strategic resource management
- Progressive difficulty scaling
- Multiple card types and rarities
- Save/load system for game progress

## Technologies Used

- JavaScript (ES6+)
- HTML5 Canvas API
- CSS3 for styling
- Local Storage for game persistence
- Procedural generation algorithms
