<div className="mt-8">
  <h3 className="text-lg font-semibold mb-3">Technical Implementation</h3>
  <ul className="list-disc list-outside space-y-2 text-sm mb-6 ml-4">
    <li>
      <strong>Hardware</strong>: Raspberry Pi 5 (8GB), ARZOPA 16" monitor,
      Logitech C920 webcam
    </li>
    <li>
      <strong>Frontend</strong>: Custom React/Electron app for mirror display
    </li>
    <li>
      <strong>Backend</strong>: Node.js/Express server with motion detection and
      API integrations
    </li>
    <li>
      <strong>AI Integration</strong>: OpenAI Vision API for outfit analysis,
      OpenAI Text-to-Speech API for voice messages, OpenAI for news headline
      summarization
    </li>
    <li>
      <strong>APIs</strong>: WeatherAPI, NewsAPI, and Google Calendar for
      personalized contextual data
    </li>
    <li>
      <strong>Mechanical</strong>: Custom 12"x18" drilled two-way mirror with
      matching clear back panel, custom foam display window, #4-40 nylon machine
      bolts and standoffs, and adjustable easel
    </li>
  </ul>
</div>

    </li>
    <li>
      <strong>Day 5</strong>: Created mobile interface with WebRTC integration (eventually scrapped)
    </li>
    <li>
      <strong>Day 6</strong>: Fundamental shift away from phone interface; purchased and integrated USB webcam
    </li>
    <li>
      <strong>Day 7</strong>: Mock demos + improvements to weather and UI
    </li>
    <li>
      <strong>Day 8</strong>: Designed full assembly; purchased easel, foam sheets, video/power cables, and mounting hardware
    </li>
    <li>
      <strong>Day 9</strong>: Text-to-speech integration
    </li>
    <li>
      <strong>Day 10</strong>: Full assembly and testing
    </li>

      <strong>Demo Day</strong>: Demo day! Set up motion detection and integration with triggering messages/interactions; configure audio messages and Ai prompts; code freeze at 4:30, then at like 7 lol
