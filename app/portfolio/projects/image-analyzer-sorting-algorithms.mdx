A visual demonstration tool that showcases various sorting algorithms through the lens of image analysis and manipulation.

## Overview

This project provides an interactive way to visualize how different sorting algorithms work by applying them to image data. Users can upload images and watch as various sorting algorithms process and reorganize the pixel data in real-time.

## Features

- Image upload and processing
- Multiple sorting algorithm implementations
- Real-time visual feedback
- Interactive controls for algorithm selection
- Performance comparison between algorithms

## Technologies Used

- JavaScript (ES6+)
- HTML5 Canvas API
- CSS3 for styling
- File API for image handling
