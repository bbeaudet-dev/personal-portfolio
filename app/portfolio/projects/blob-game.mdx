An engaging blob simulation game where players control a growing blob that can consume smaller blobs and avoid larger ones in a dynamic environment.

## Overview

This game features a blob that grows as it consumes smaller blobs while avoiding larger predators. The blob uses realistic physics for movement and collision detection, creating an immersive gaming experience.

## Features

- Realistic blob physics and movement
- Dynamic growth and shrinking mechanics
- Collision detection system
- Multiple difficulty levels
- Score tracking and progression
- Smooth animations and visual effects

## Technologies Used

- JavaScript (ES6+)
- HTML5 Canvas API
- CSS3 for styling
- RequestAnimationFrame for smooth animations
