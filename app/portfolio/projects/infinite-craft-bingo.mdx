A creative bingo game inspired by Infinite Craft, where players mark off crafting combinations and elements as they're discovered, combining the addictive nature of crafting games with classic bingo mechanics.

## Overview

This project takes the concept of Infinite Craft's element combinations and transforms it into a bingo format, where players track discovered combinations and compete to complete their bingo cards with various crafting elements and recipes.

## Features

- Infinite Craft-themed bingo cards
- Element combination tracking
- Multiple difficulty levels
- Combination discovery system
- Progress tracking and statistics
- Customizable card layouts
- Achievement system for rare combinations
- Social sharing of completed cards

## Technologies Used

- JavaScript (ES6+)
- HTML5 semantic markup
- CSS3 with responsive design
- Local Storage for progress
- Event-driven programming
- JSON data structures for combinations
